# 🚀 Codebase | Python & C#

Chào mừng bạn đến với kho lưu trữ mã nguồn cá nhân. Đây là nơi tổng hợp các dự án, công cụ và bài tập thực hành mà tôi đã xây dựng trong quá trình phát triển kỹ năng lập trình.

**Quyền sử dụng:** Bạn có thể sử dụng mã nguồn của tôi thoải mái cho mục đích học tập hoặc cá nhân.

Để giữ cho kho lưu trữ gọn gàng và dễ theo dõi, các dự án được phân loại theo ngôn ngữ lập trình chính.

## 📁 Danh mục dự án

Bạn có thể truy cập vào các thư mục dưới đây để xem chi tiết mã nguồn và tài liệu của từng dự án:

*   **[`/python-projects/`](./python-projects/)**: Các dự án sử dụng Python (Automation, Data, Scripts, Web, v.v.)
*   **[`/csharp-projects/`](./csharp-projects/)**: Các dự án sử dụng C# (.NET, Desktop Apps, Game, v.v.)

> **Lưu ý:** Mỗi thư mục con đều chứa các tệp tin hướng dẫn riêng biệt. Bạn có thể nhấn vào từng thư mục để xem chi tiết mô tả và cách thức hoạt động của từng dự án cụ thể.
